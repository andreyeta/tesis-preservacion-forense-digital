# Procesamiento de Datos para Tesis de Maestría

## Descripción

Este repositorio contiene los scripts, conjuntos de datos y resultados utilizados para el procesamiento y análisis de información correspondiente a la investigación:

**"Análisis del nivel de implementación de prácticas de preservación forense de la información digital en la Dirección de Sistemas de Información del Ministerio de Desarrollo Social (MIDES)"**

## Objetivo

Desarrollar un proceso automatizado para:

- Procesamiento de datos de listas de verificación.
- Generación de indicadores de cumplimiento.
- Clasificación de niveles de implementación.
- Elaboración de estadísticas descriptivas.
- Generación automática de gráficos.
- Preparación de datos para Power BI.

## Herramientas utilizadas

- Python 3.x
- Pandas
- Matplotlib
- OpenPyXL
- Microsoft Excel
- Power BI

## Resultados generados

- Resúmenes estadísticos.
- Indicadores de cumplimiento por dimensión.
- Gráficos de análisis.
- Dataset preparado para visualización en Power BI.

## Estructura metodológica

El análisis se desarrolló siguiendo las fases:

1. Recolección de datos mediante checklist institucional.
2. Transformación y validación de datos.
3. Cálculo de indicadores de cumplimiento.
4. Clasificación por niveles (Alto, Medio y Bajo).
5. Generación de visualizaciones.
6. Elaboración de resultados para la tesis.
